1. 我要制作一个犬和猫的每日摄入卡路里计算器

2. 公式步骤
  1. 先计算出静息能量需求 RER (这是宠物在常温、安静状态下维持基本生命所需的热量) : RER（千卡/天） = 70 × 体重(kg)^0.75
  2. 再计算出每日代谢能量需求 MER（系数取决于宠物的生命阶段和活动量。犬猫系数不同，请根据下方信息取系数）： MER = RER × 系数
3. 犬类系数信息，依次按选择确定系数，下一个问题选项的答案对应的覆盖上一个问题选项的系数，除非个别选项备注延续上一个选项的系数
  1. 性别 （无系数，用于决定是否展现孕期选择）
    1. 男宝
    2. 女宝
  2. 绝育状态
    1. 已绝育：1.6
    2. 未绝育：1.8
  3. 运动时长（基于绝育状态选项校准）
    1. 轻度运动（每天小于1小时）：1.2（若绝育状态选已绝育），1.4（若绝育状态选未绝育） 
    2. 中度运动（1~3小时/天）：若绝育状态选已绝育则系数保持1.6，未绝育则系数保持1.8
    3. 高强度工作/赛犬：2
  4. 体重情况
    1. 严重超重：1
    2. 超重：1.2
    3. 正常体型：保持运动时长的系数
    4. 偏瘦：2
  5. 犬类年龄
    1. 宝宝（4个月以下）：3.0
    2. 幼犬（4~12个月）：2.0
    3. 成年犬（大于等于1岁小于7岁）：延续体重情况的系数
    4. 老年（>=7岁）：1.2
  6. 怀孕和哺乳情况（性别选女宝，绝育状态选未绝育，犬类年龄选不是宝宝犬和幼犬才会展示该问题）
    1. 怀孕期：1.8
    2. 哺乳期：2.0
    3. 不在怀孕和哺乳期：保持犬类年龄选项系数
  7. 狗狗是否在术后恢复期
    1. 绝育手术后一周内：1.4
    2. 其他重症术后一周内：1
    3. 不在术后期：保持犬类年龄选项系数


4. 猫类系数信息，依次按选择确定系数，下一个问题选项的答案对应的覆盖上一个问题选项的系数，除非个别选项备注延续上一个选项的系数
  1. 性别 （无系数，用于决定是否展现孕期选择）
    1. 男宝
    2. 女宝
  2. 绝育状态
    1. 已绝育：1.2
    2. 未绝育：1.4
  3. 是否户外散养（基于绝育状态选项校准）
    1. 是：1.6
    2. 否：延续绝育状态的系数
  4. 体重情况
    1. 是：0.9
    2. 正常：延续否户外散养的系数
    3. 偏瘦：1.4
  5. 猫咪年龄
    1. 宝宝（4个月以下）：2.7
    2. 幼猫（4~12个月）：2.0
    3. 成年猫（大于1岁小于7岁）：延续体重情况的系数
    4. 老年（>=7岁）：1
  6. 怀孕情况（性别选女宝，绝育状态选未绝育，犬类年龄选不是宝宝犬和幼犬才会展示该问题）
    1. 是：2.5
    2. 否：延续猫咪年龄的系数
  7. 猫咪是否在术后恢复期
    1. 绝育手术后一周内：1.2
    2. 其他重症术后一周内：1
    3. 不在术后期：系数继承
